# 行为准则

## 我们的承诺

为建设开放、友好的技术社区，我们承诺：无论年龄、体型、可见或不可见残障、族群、性别特征、性别认同与表达、经验水平、教育、社会经济地位、国籍、相貌、种族、宗教或性取向，参与本组织项目的人都能免于骚扰。

## 行为标准

**鼓励的行为**包括：

- 善意、尊重不同观点与经验  
- 建设性反馈，对事不对人  
- 关注对社区与项目最有利的结果  
- 负责任地披露安全问题（见 [SECURITY.md](./SECURITY.md)）

**不可接受的行为**包括：

- 性化语言或意象、不受欢迎的性关注  
- 钓鱼、侮辱、贬损性评论、人身或政治攻击  
- 公开或私下骚扰  
- 未经许可发布他人隐私信息  
- 其他在专业场合可被合理视为不当的行为  

## 执行

不当行为可报告至组织维护者（GitHub org Owner 或 `security@f2b.dev` / 组织公开联系邮箱，标题 `[CONDUCT]`）。维护者将审查并采取其认为适当的措施，包括警告、临时或永久限制参与。

本准则适用于所有项目空间及代表项目的公共场合。
